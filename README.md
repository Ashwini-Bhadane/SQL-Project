# Employee SQL Project

A complete SQL mini-project containing:
- Table creation
- Sample data insertion
- Joins
- Group By + Having
- Aggregate function queries
- Subqueries (max, second max, above average salary)

## Files
- `1_create_tables.sql` – All table creation
- `2_insert_data.sql` – Insert sample employees & departments
- `3_queries.sql` – All SQL practice queries
- `README.md` – Project description

## Features Covered
- Inner Join, Left Join
- Count employees in each department
- Total salary per department
- Average salary per department
- Departments with more than 3 employees
- Highest salary / Second highest salary (subqueries)
- Employees earning above average salary
